# 2. 常用过滤条件

- 目的IP

```
ip.dst == 10.24.58.22   筛选目的IP是10.24.58.22 的包
```
- 源IP

```
ip.src == 10.24.58.22   筛选源IP是10.24.58.22 的包
```
- 多个条件： and（与），or（或）
